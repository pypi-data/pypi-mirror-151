# Puddler (Emby/Jellyfin-MPV-CLI)
Emby/Jellyfin command line client, powered by mpv.

### Currently, in extreme buggy alpha state.
___

### Installation:
```
$ python -m pip install Puddler --upgrade
$ python -m puddler
```
